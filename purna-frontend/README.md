# Purna Meal Planner

A meal planning app built with Vite, React, TypeScript, and shadcn-ui.

## Prerequisites

- **Node.js** (v18 or later recommended) and **npm**
- Install Node via [nvm](https://github.com/nvm-sh/nvm#installing-and-updating) if needed

## How to run the project

### 1. Clone the repository

```sh
git clone https://github.com/muskan149/purna-meal-planner.git
cd purna-meal-planner
```

### 2. Install dependencies

```sh
npm install
```

### 3. Start the development server

```sh
npm run dev
```

The app will be available at the URL shown in the terminal (usually `http://localhost:5173`). The dev server supports hot reload.

### Other useful commands

| Command           | Description                    |
|-------------------|--------------------------------|
| `npm run build`   | Build for production           |
| `npm run preview` | Preview the production build   |
| `npm run lint`    | Run ESLint                     |
| `npm run test`    | Run tests                      |

## Tech stack

- **Vite** – build tool
- **React** – UI
- **TypeScript** – typing
- **shadcn-ui** – components
- **Tailwind CSS** – styling
