# Purna Backend API

Backend API for the Purna meal planner application.

## Live API

- **Base URL:** https://purna-backend.vercel.app/
- **API docs (Swagger):** https://purna-backend.vercel.app/docs
- **Health check:** https://purna-backend.vercel.app/health

## Overview

This directory is the home for backend-related code and documentation. The API is deployed on Vercel.

Use the [API docs](https://purna-backend.vercel.app/docs) for endpoint reference and testing.
